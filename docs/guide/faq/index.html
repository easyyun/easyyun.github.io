如何申请app key​

公众号申请 

格式如下
标题：申请API
单位名称（或个人）：
姓名：
电话：
功能（需要哪些接口）：
提示

线上文档除token无法设置外，大部分接口均可以直接测试，请先自行测试，评估需要哪些接口，然后再申请API（朝九晚五时间内申请）。 本着节省资源原则，申请API超过一周未使用，将自动过期收回。

PS:（Word、PPT、Excel）类相关转换未开放，建议使用其他厂商的商业产品。
