# FAQ



## 如何申请app key 

发送邮件 kf@shudengyun.com ，格式如下

```
申请app key 
联系电话：
单位名称：
用途：
```


## 测试环境与正式环境差异

| 类别 | 测试 | 正式 |
| --- | --- | --- |
| 文件大小 | 3M | 10M |
| 请求频率 | 每秒1次 | 无限制 |
| 文件有效期 | 1小时后销毁 | 程序自行控制 |
| 文件存储服务商 | 阿里云 | 阿里云、华为、腾讯 |
| 特殊API | 不支持，部分接口有限制 | 支持 |
